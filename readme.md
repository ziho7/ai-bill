AI记账软件全栈开发项目

功能
1. 登录注册
2. 对话功能
  - 普通对话
  - 理解自然语言转换成账单
3. 首页数据展示，并且可以切换不同的日期

技术栈
移动端: 
  - 框架: React native (Expo)
  - 状态管理: Zustand
后端:
  - Nodejs框架Express + Typescript
  - 登录注册: Supabase auth + JWT
  - 对话功能实现: AI SDK
  - 数据库: Supabase提供的Postgresql
  - ORM: Drizzle
  - 模型: Deepseek
